使用2D Toolkit Extensions步骤：
1.在http://www.unikronsoftware.com/2dtoolkit网站或者Unity Asset Store下载最新版本的2D TOOLKIT
2.导入2D TOOLKIT到工程
3.在Unity Asset Store下载2D Toolkit Extensions(简称2D Toolkit Ex)
4.导入2D Toolkit Ex到工程
5.删除TK2DROOT/tk2dUI/Code/Controls/tk2dUIScrollableArea.cs文件

帮助：
1.IconSpriteManager功能同cocos2dx的TextureCache
2.tk2dGridView功能同iOS UITableView，但增加了多列功能
